# Análise dos Livros Mais Vendidos da Amazon (2009–2019)

Este projeto apresenta uma análise exploratória dos livros mais vendidos na Amazon por ano, entre 2009 e 2019, utilizando Power BI para visualização de dados.

---

## Objetivo

O objetivo principal desta análise é identificar padrões e tendências no mercado editorial, respondendo perguntas como:

* Um preço mais alto garante melhor avaliação?
* Existem diferenças entre livros de Ficção e Não Ficção?
* Quais autores aparecem com mais frequência na lista de best-sellers?
* Popularidade está relacionada à qualidade?

---

## Dataset

O conjunto de dados contém:

* 550 registros
* Período: 2009 a 2019
* Categorias: Ficção e Não Ficção

---

## Ferramentas Utilizadas

*  **Power BI**
*  **DAX (Data Analysis Expressions)**
*  Excel / CSV

---

## Principais Insights

* Livros de **Não Ficção são mais vendidos e tendem a ter preços mais altos**, contudo livros de Ficção possuem melhor avaliação entre os usuários.
* **Preço não tem forte correlação com avaliação**.
* Popularidade nem sempre indica maior qualidade.

---

## Conclusão

Este projeto demonstra como técnicas de análise de dados e visualização podem revelar padrões relevantes no mercado editorial, auxiliando na tomada de decisões e entendimento do comportamento do consumidor.
